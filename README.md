## 指标性质验证
1. 框架
    * Flask
    * html
2. 需求
    * 异步
    * 多进程计算
    * 完成指标间比较
    
### 细分模块
1. 指标输入
    * 输入格式校验，返回对应的错误说明
    * 判断当前指标频率
2. 计算配置参数设计
    * 特征性质：相似性、滞后性、逻辑性（因果关系）
    * 搜寻区间：近3年、近6年、近10年、全部时间
    * 特征频率：同频、混频、同频+混频
    * 输出特征排名：连续值
3. 数据处理
    * 根据配置参数生成数据库数据抽取的sql
    * 保存成数据库配置文件
4. 计算
    * 不同特征性质下的计算方法
    * 中间结果保存
5. 结果输出
    * 指标特征性质排名
    * 形态输出
    * 指标位置信息
6. 其他
    * 异步
    * 前端
    * 用户管理
